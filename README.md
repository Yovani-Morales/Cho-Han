# Cho Han 🎮🎲
Es una **Demo** de un juego de apuestas japonés que consiste en agitar dos dados en una taza y la suma de ambos da como resultado Cho (par) o Han (impar). [Wikipedia](https://en.wikipedia.org/wiki/Ch%C5%8D-han)
